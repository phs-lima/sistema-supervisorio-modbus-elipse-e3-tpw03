# Sistema Supervisório Industrial com Elipse E3, Modbus, Arduino e CLP TPW03

## Sobre o Projeto

Projeto desenvolvido durante o curso de Tecnologia em Automação Industrial com o objetivo de integrar um sistema supervisório desenvolvido no Elipse E3 a um CLP TPW03 utilizando comunicação Modbus através de um Arduino.

O sistema permitia monitoramento e controle de dispositivos industriais por meio de uma Interface Homem-Máquina (IHM), possibilitando a operação remota de cargas e a visualização de variáveis do processo em tempo real.

## Arquitetura do Sistema

Elipse E3 (IHM/SCADA)
        ↓
Comunicação Modbus
        ↓
Arduino
        ↓
CLP TPW03
        ↓
Dispositivos de Campo

- Lâmpada
- Sensor de Temperatura
- Motor Trifásico

---

## Funcionalidades Implementadas

### Monitoramento

- Leitura da temperatura em tempo real.
- Exibição das variáveis do processo na IHM.
- Monitoramento do estado dos dispositivos.

### Controle

- Acionamento remoto de iluminação.
- Partida e parada de motor trifásico.
- Envio de comandos através da IHM.

### Comunicação

- Integração entre Elipse E3 e Arduino.
- Comunicação Modbus.
- Troca de dados entre Arduino e CLP TPW03.
